Abstract

This project explores the vulnerabilities of monocular depth estimation (MDE) models to adversarial patch attacks in autonomous driving. We use the 3D2Fool algorithm to craft optimized 2D adversarial patches and test them on SOTA models like Monodepth2, DenseDepth, ZoeDepth, and DepthAnythingV2. We introduce adversarial training to defend these models and restore robustness in predictions.

Sample Visualizations

Methodology

The CARLA simulator is used to generate a synthetic dataset. YOLOv8 is applied for car detection, followed by optimized patch application and training using DenseDepth. A custom loss function maximizes disruption to depth maps in the patched region.

üîê IDS for Internet of Vehicles using ML, DL and Ensemble Methods

Abstract

This project presents a comprehensive comparison of machine learning, deep learning, and ensemble learning algorithms for intrusion detection in the Internet of Vehicles (IoV). Using the CICIoV2024 dataset, we benchmark models such as Random Forest, Logistic Regression, AdaBoost, and LSTM on binary, 3-class, and multi-class classification tasks.

Methodology

Data preprocessing involved missing value treatment, encoding, and balanced train-test-validation splits with stratification. Models were tested on binary and decimal forms of the dataset. An ensemble model was created using max-voting across RF, LR, and AB.
